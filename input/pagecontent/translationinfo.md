# Translation information

This implementation guide is published in multiple languages. Narrative pages in Lithuanian are translated from the English source; some labels and template text come from the FHIR IG Publisher and dependent packages.

For general guidance on multi-language implementation guides, see [Producing Multi-Language IGs](https://build.fhir.org/ig/FHIR/ig-guidance/languages.html) in the FHIR IG guidance.
